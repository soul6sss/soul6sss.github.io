# 🌐 个人网站设计文档 (Vue)

## 1. 整体风格

- **主题**：科技感 / 简洁炫酷
- **色彩**：深色背景 + 渐变点缀（蓝紫、青绿、橙黄等霓虹色系）
- **排版**：模块化分区（Hero、About、Projects、Contact），使用全屏滚动或锚点跳转
- **字体**：标题字体粗犷、现代感；正文保持易读

------

## 2. 页面结构与组件设计

### (1) `HeaderNav.vue` – 顶部导航栏

- 固定在页面顶部，透明背景 → 滚动时变为半透明黑色
- 包含锚点链接：**Home / About / Projects / Contact**
- 移动端收缩为汉堡菜单

**交互逻辑**：

- 点击菜单自动平滑滚动到对应 section
- 当前所在模块高亮显示

------

### (2) `HeroSection.vue` – 首页/个人介绍

- 背景：粒子动画 / 星空 / 流动渐变
- 居中显示个人姓名、大标题、打字机效果的职业标签（如“前端开发者 / AI 爱好者”）
- 一个 “查看更多” 的按钮，点击滚动到 About 区域

**交互逻辑**：

- 鼠标悬停在名字上 → 光晕效果
- 点击按钮 → 平滑滚动

------

### (3) `AboutSection.vue` – 关于我

- 左侧：圆形头像（呼吸灯环绕动画）
- 右侧：简介文字，技能标签（Vue / JS / Python / AI 等图标）
- 技能栏：用进度条或环形图表示熟练度

**交互逻辑**：

- 鼠标悬停头像 → 放大/发光
- 技能标签 hover → 上浮 + tooltip 显示熟练度

------

### (4) `ProjectsSection.vue` – 作品集

- 卡片布局（3 列 / 2 列自适应），每个卡片包含：
  - 缩略图
  - 项目名称
  - 技术栈标签
- 鼠标悬停：卡片放大，显示遮罩层和“查看详情 / GitHub”按钮

**交互逻辑**：

- 点击卡片 → 弹出 Modal（展示更详细的描述 + 截图 + Demo 链接）
- 支持按标签过滤（前端 / AI / 工具类）

------

### (5) `ContactSection.vue` – 联系方式

- 背景渐变或地图样式
- 包含：邮箱、GitHub、LinkedIn、微信二维码
- 一个简单留言表单（姓名 / 邮箱 / 留言），表单提交到 Formspree

**交互逻辑**：

- 按钮 hover → 渐变闪烁
- 表单验证（邮箱格式、内容必填）

------

### (6) `Footer.vue` – 页脚

- 简洁版权信息 + 社交媒体图标
- 可加一句座右铭或签名

------

## 3. 动效设计

- **进入动画**：滚动到某个模块时淡入/滑入 (`AOS` 或 `Framer Motion`)
- **按钮动效**：hover 渐变、按下缩放
- **背景特效**：粒子、星空、彩色渐变波浪
- **作品卡片**：悬停上浮 + 投影增强

------

## 4. 用户体验 (UX) 设计

- **自适应**：移动端一列布局，导航变折叠菜单
- **平滑滚动**：点击导航或按钮时，页面平滑跳转
- **亮/暗模式切换**（可选）
- **加载优化**：懒加载图片、压缩资源
