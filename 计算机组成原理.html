第一章 计算机系统概论

一、 计算机系统的基本组成

总纲： 计算机系统 = 硬件系统 + 软件系统

1. 软件系统

软件按其功能和角色分为两大类：

  • ① 系统软件

    • 定义： 用于管理计算机资源，管理硬件并提供运行环境。

    • 典型代表：

      • 操作系统（Windows, Linux）

      • 编译器

      • 数据库系统

      • 驱动程序

    • 作用： 管理硬件 + 提供运行环境

  • ② 应用软件

    • 定义： 用户为解决具体问题而使用的软件。

    • 典型代表： 浏览器、游戏、Office、微信等。

二、 冯·诺依曼计算机结构

现代计算机的祖先结构，由 冯·诺依曼 于 1945 年提出。

1. 五大核心思想（重点）

  1. 使用二进制： 计算机内部用 0 和 1 表示数据。

  2. 存储程序思想（最核心）：

    • 程序和数据统一存放在存储器中。

    • 不是接线控制程序，程序也作为数据存储。

  3. 五大部件结构： 运算器、控制器、存储器、输入设备、输出设备。

  4. 指令顺序执行： 程序一般按顺序执行，除非遇到跳转指令。

  5. 指令和数据同等对待： 程序也是数据。

结构示意： 输入 → 存储器 ↔ CPU → 输出 （CPU从存储器取指令和数据）

2. 冯·诺依曼结构的缺点（关键考点）

  • 最大问题： 冯·诺依曼瓶颈 (存储器瓶颈)

  • 原因： 指令 和 数据 共用一个存储器。

  • 后果： 同一时间不能同时取指令和数据，导致速度受限。

三、 现代计算机结构（改进点）

现代计算机仍基于冯·诺依曼，但针对瓶颈做了改进。

主要改进点

  1. CPU内部并行处理： 增加 Cache、多级流水线、多核，提高速度。

  2. 指令和数据分离（哈佛结构思想）：

    • 现代CPU中常见：指令Cache 和 数据Cache 分离。

    • 效果： 可以同时取指令 + 取数据，速度大幅提升。

  3. 多级存储体系：

    • 结构：寄存器 ↓ Cache ↓ 主存 ↓ 硬盘

四、 计算机性能指标与计算

1. 什么叫计算机性能？

  • 核心定义： 完成任务所需时间越短，性能越好。

  • 核心指标： ✅ 运行时间（执行时间）

两种时间概念：

  1. 响应时间（周转时间）： 用户等待时间（含 CPU计算 + IO等待 + 调度）。

  2. CPU时间（考试重点）： 真正CPU执行时间（CPU用于执行程序的时间，不包括 IO等待）。

2. 最核心公式（必须掌握）

CPU时间=指令条数×CPI×时钟周期

或者写作：

CPU时间=指令条数(IC)×CPI时钟频率(f)

各部分含义：

  • 指令条数 (IC)： 程序执行的总指令数。

  • CPI (Cycles Per Instruction)： 每条指令平均周期数（越小越好）。

  • 时钟周期： CPU节拍时间 (1/主频)。

完整计算举例：

  • 假设：指令数 = 109，CPI = 2，主频 = 2GHz

  • 计算：CPU时间=109×22×109=1 秒

3. 常见性能指标

  1. 主频（时钟频率）：

    • 单位：GHz, MHz。表示 CPU 每秒的时钟次数。

    • 注意： 主频高 ≠ 性能一定高（还要看 CPI）。

  2. CPI：

    • 每条指令需要多少周期。

    • 平均CPI=∑(各类指令数×各类CPI)总指令数

  3. MIPS（百万指令每秒）：

    • 公式：MIPS=主频CPI×106

    • 缺点：不同指令复杂度不同，比较不公平。

  4. FLOPS（浮点运算速度）：

    • 每秒浮点运算次数 (TFLOPS, PFLOPS)。常用于超算、AI。

  5. 吞吐量：

    • 单位时间完成任务数量。

五、 考试重点与记忆技巧

1. 最容易考的计算题类型

  • 类型① 比较性能：

    • 性能A性能B=时间B时间A

  • 类型② 改变部件后的性能 (Amdahl定律)：

    *

    • 加速比=1(1−改进部分)+改进部分加速倍数

  • 类型③ 求CPI或时间：

    • 给指令比例、CPI、主频，求执行时间。

2. 整章核心记忆结构

程序 → 指令数(IC) → CPI → CPU周期数 → 主频 → 执行时间

指令多少，

周期几遍，

一、什么是定点数

定点数（Fixed-point number）： 指小数点位置固定不变的数。

也就是说：

  • 小数点位置是约定好的

  • 存储时只保存数字本身

  • 默认某个位置就是小数点
